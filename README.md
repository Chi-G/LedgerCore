## 🚀 Project Overview

This repository contains the official website and client portal for Forahia Solutions, built with modern web technologies to showcase our services, portfolio, and provide tools for our clients.

### Key Features
- **Interactive Portfolio Showcase** - Dynamic project filtering and case studies
- **ROI Calculator** - Personalized digital transformation ROI projections
- **Client Portal** - Resources, testimonials, and support tools
- **Leadership Team Profiles** - Meet our global team of experts
- **Innovation Lab** - R&D initiatives and open-source contributions
- **Responsive Design** - Optimized for all devices and screen sizes

## 🛠️ Technology Stack

### Backend
- **Laravel 12** - PHP 8.2+ framework
- **Inertia.js** - Seamless SPA experience
- **MySQL** - Database management

### Frontend
- **Vue.js 3** - Progressive JavaScript framework
- **Tailwind CSS 4.x** - Utility-first CSS framework
- **Vite** - Fast build tool and dev server

### Development Tools
- **Pest** - PHP testing framework
- **Laravel Pint** - Code style fixer
- **Concurrently** - Parallel task runner
